Sorun Giderme

  • 1. Tarayıcı Desteği: Chrome, Edge veya Firefox kullanın. Safari'de Gamepad API desteği sınırlıdır.
  • 2. HTTPS Bağlantısı: Bu sayfayı HTTPS üzerinden açın veya localhost'tan çalıştırın.
  • 3. Gamepad'i Etkinleştir: Gamepad bağlıyken herhangi bir butona basın.
  • 4. Tarayıcı Ayarları: Chrome'da chrome://flags/#enable-gamepad-button-axis-events adresine giderek Gamepad desteğini etkinleştirin.
  • 5. Sürücüler: Gamepad sürücülerinizin güncel olduğundan emin olun.

Bağlantı Problemi Çözümleri

  • 1. Butona Basın: Gamepad bağlıyken herhangi bir butona veya tetiğe basın.
  • 2. Chrome Kullanın: Chrome'da chrome://gamepad yazarak test edin.
  • 3. HTTP/HTTPS: Bu sayfayı http://localhost'ta açın veya HTTPS kullanın.
  • 4. Gamepad'i Çıkarıp Takın: USB gamepad'i çıkarıp yeniden takın.
  • 5. Bluetooth: Bluetooth gamepad için sistem ayarlarından eşleştirin.
  • 6. Farklı Port: USB portunu değiştirmeyi deneyin.
